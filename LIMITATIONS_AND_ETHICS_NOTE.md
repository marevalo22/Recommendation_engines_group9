# Limitations and Ethics Note

The synthetic layer is useful for proof-of-concept quality, tactical explanation, and scenario modelling, but it comes with clear limitations.

1. **The real-only benchmark remains the only honest core benchmark.** It should remain the primary evaluation in the report and presentation.
2. **The synthetic PoC track is label-informed.** It is helpful for demonstrating how richer scouting environments can improve ranking and explanation, but it is not a fair estimate of real-world out-of-sample performance.
3. **Public football data remains incomplete.** Actual transfer decisions depend on contracts, agents, wages, work permits, injuries, manager preferences, and personal factors that are not fully captured here.
4. **Synthetic variables are stylised proxies.** They are designed to be role-aware, team-aware, and plausible, but they are still engineered constructs rather than directly observed event or tracking data.
5. **Shortlist plausibility still needs human judgement.** Even with affordability and squad-fit features, a final transfer board should still be reviewed by domain experts.

Recommended wording for the final submission:

> Real-only results are treated as the honest benchmark. Synthetic-augmented results are presented separately as proof-of-concept / scenario-modelling outputs that improve tactical richness and explainability, but do not represent unbiased real-world generalisation.
